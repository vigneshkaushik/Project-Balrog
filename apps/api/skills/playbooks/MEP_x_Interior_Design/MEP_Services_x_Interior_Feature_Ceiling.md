## Anchor constraint

Feature ceilings (coffered, curved, timber baton, metal mesh, stretched fabric, GRG, backlit) are interior design flagship elements that command significant cost and fabrication lead time. They cannot be compromised by visible MEP intrusions, inadequate access, or incompatible service positions. MEP must be fully resolved and frozen before feature ceiling design is finalised.

## Recommended resolutions (ranked)

### 1. Complete MEP coordination above ceiling zone before ceiling is designed

**Effort:** Medium.

All MEP services (ducts, pipes, sprinklers, cable trays) must be fully coordinated and frozen within the plenum above the feature ceiling level before the ceiling design is finalised. The ID team receives a confirmed MEP soffit plan showing exact service positions, and designs the ceiling module, access panel locations, and service outlet positions (diffusers, grilles, pendants) around the confirmed MEP layout.

**Checks:**

- [ ] MEP services fully coordinated in zone before feature ceiling design is locked
- [ ] All sprinkler heads, diffusers, and grilles positioned within ceiling module before design is finalised
- [ ] All maintainable MEP items (valves, J-boxes, dampers) above ceiling have designed access panel positions

### 2. Design ceiling access strategy as an architectural feature

**Effort:** Medium.

Feature ceilings must include access to all maintainable MEP above them. For lift-out ceilings: removable panels or hinged sections at every accessible valve, junction box, and damper. For fixed ceilings (plasterboard, timber, GRG): access panels must be designed as architectural features — flush-mounted, painted or veneered to match, or with an expressed reveal. All access panel positions are shown on the ID drawing set and approved by the ID team before fabrication.

**Checks:**

- [ ] Every maintainable MEP item above feature ceiling has an associated access panel
- [ ] Access panels designed and approved by ID before ceiling fabrication order is placed
- [ ] Panel sizes meet trade contractor minimum access requirements: 600×600mm for body access, 300×300mm for small valve isolation

### 3. Relocate all maintainable MEP outside the feature ceiling boundary

**Effort:** High.

If feature ceiling is non-accessible (fixed, curved, artistic), the MEP engineer must redesign service routes so that no maintainable elements (valves, junction boxes, dampers) are located within the feature ceiling boundary. Only self-sealing or non-maintainable items (fixed sprinkler heads) may remain. This is a significant MEP redesign scope.

**Checks:**

- [ ] Confirmed no maintainable MEP within inaccessible feature ceiling zone
- [ ] MEP engineer has redesigned routes to comply
- [ ] Fire authority confirmation that sprinkler positions within feature ceiling are code-compliant

## Validate before acting

- [ ] Obtain feature ceiling design drawings including boundary, height, and module before MEP coordination
- [ ] Confirm whether feature ceiling is accessible (lift-out tiles) or fixed (GRG, timber, stretched fabric)
- [ ] Identify all MEP elements in the feature ceiling zone that require maintenance access

## Escalate immediately if

- Feature ceiling is fixed, non-accessible, and has already been fabricated — any MEP modification will require ceiling damage and repair at significant cost
- Sprinkler head positions within the feature ceiling do not meet fire authority spacing requirements — fire engineering performance solution required

## Downstream risk if unresolved

Feature ceilings are the highest-value, longest-lead interior elements. Any clash discovered after fabrication or installation is a major cost and programme catastrophe. Ensure MEP is fully coordinated and signed off — in writing by all parties — before the feature ceiling fabrication package is released for production.
