## Anchor constraint

PT slabs contain live tensioned tendons that are lethal to cut and will structurally compromise the slab if severed. ALL penetrations in PT slabs require SE approval with exact tendon layout confirmed. This is a zero-tolerance safety item.

## Recommended resolutions (ranked)

### 1. Relocate penetration to SE-approved safe zone

**Effort:** Medium.

SE must provide a PT tendon layout drawing with pre-approved penetration zones marked. MEP penetrations must be relocated to land within approved zones: typically column-to-column midpoint (maximum tendon sag), banded slab areas, and specific pre-formed openings. Never core-drill a PT slab without SE confirming tendon-free zone via GPR scan at that exact location.

**Checks:**

- [ ] PT tendon layout drawing obtained from SE
- [ ] GPR scan of target area commissioned before any coring
- [ ] Written SE confirmation that proposed location is tendon-free

### 2. Pre-form opening before concrete pour

**Effort:** Low.

If PT slab is not yet poured, coordinate MEP penetration requirements immediately and pre-form all openings in formwork at SE-approved locations. This is the lowest cost and lowest risk option. MEP engineers must finalise penetration positions before pour date — no exceptions.

**Checks:**

- [ ] All MEP penetration requirements finalised before pour date
- [ ] SE to approve and mark all openings on formwork drawings
- [ ] Contractor pour programme reviewed — penetration coordination must complete first

### 3. GPR scan + core drill in confirmed safe zone (post-pour)

**Effort:** High. **VO risk:** Yes (formal variation / redesign likely).

If slab is already poured and opening was missed: commission a GPR scan from a certified operator. SE reviews GPR results and provides written approval (or denial) for coring at the specific location. Core drill ONLY after written SE approval. Document with photographic record of entire process.

**Checks:**

- [ ] GPR scan from certified operator
- [ ] SE written approval before any drilling commences
- [ ] Drilling contractor must not deviate from approved location by more than 25mm

## Validate before acting

- [ ] Confirm slab construction type from structural drawings before ANY clash resolution
- [ ] Request PT tendon layout plan from SE as the first step — before exploring any resolution options
- [ ] Check pour programme dates — if slab not yet poured, pre-forming is ALWAYS the preferred option

## Escalate immediately if

- Any field coring of a PT slab without documented SE approval and GPR scan — STOP WORK
- A tendon has been struck during drilling — emergency structural assessment required immediately and work halted

## Downstream risk if unresolved

Severed PT tendons cannot be repaired in situ. The structural consequence ranges from local cracking to slab collapse. This clash must be resolved at design stage. Escalate to SE and PM the moment this clash is identified.
