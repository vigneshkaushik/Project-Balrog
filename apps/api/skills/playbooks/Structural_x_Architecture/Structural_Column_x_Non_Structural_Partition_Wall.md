## Anchor constraint

Structural column is immovable. The partition wall position is the flexible element. Resolution should always produce a clean architectural outcome — boxing, wrapping, or featuring the column is preferable to an awkward partition notch.

## Recommended resolutions (ranked)

### 1. Wrap partition around column as a pilaster

**Effort:** Low.

Extend the partition to wrap around the column face, creating a pilaster or boxed-out column feature. Maintain minimum 25mm clearance from column face to inner face of boxout (fire protection build-up + movement tolerance). Coordinate electrical outlets and data points in the partition — they cannot be on the boxout face without careful MEP coordination.

**Checks:**

- [ ] Boxout dimensions allow for column fire protection + 25mm clearance
- [ ] MEP outlets/switches in partition not stranded behind the column boxout
- [ ] Boxout material and finish matches partition spec

### 2. Offset partition to clear column with expressed column in one space

**Effort:** Low.

Move the partition face to one side of the column, leaving the column partially exposed in the adjacent space. Update room dimensions on architectural drawings. Confirm minimum room width code requirements are still met. Verify interior designer accepts exposed column in adjacent space.

**Checks:**

- [ ] Minimum room dimensions maintained after offset
- [ ] ID/architect approval for exposed column face in adjacent space
- [ ] MEP services in partition re-checked after offset

### 3. Column as room feature with designed casing

**Effort:** Low.

Design the column as an architectural feature — exposed with a designed finish (painted, timber-clad, stone-clad). Eliminates the partition clash at this location. Only applicable where the room layout permits the column to be freestanding within the space.

**Checks:**

- [ ] Architect/ID approval for exposed column treatment and finish spec
- [ ] Column casing material does not compromise fire protection rating

## Validate before acting

- [ ] Confirm column is structural — some "columns" in architectural drawings are decorative fins
- [ ] Identify column fire protection method — affects minimum boxout dimensions
- [ ] Check any MEP services (conduits, pipes) planned to run in the partition at the column location

## Escalate immediately if

- Partition is a fire-rated compartment boundary wall — column boxout detail must maintain the full fire rating of the wall without compromise

## Downstream risk if unresolved

Column-partition clashes caught during fitout will cascade into joinery revision if built-in furniture is planned against the same partition. Resolve before fitout design is finalised — not during fitout installation.
