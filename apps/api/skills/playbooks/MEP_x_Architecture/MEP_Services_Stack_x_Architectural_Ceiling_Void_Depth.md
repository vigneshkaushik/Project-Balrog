## Anchor constraint

Ceiling level is architecturally and spatially governed. MEP services must fit within the plenum void above. If the void is insufficient, the ceiling must be lowered OR the services must be redesigned. Neither trade can resolve this alone — it requires a joint decision.

## Recommended resolutions (ranked)

### 1. Produce a coordinated services section (CSS) drawing for the zone

**Effort:** Medium.

This clash is symptomatic of a congested zone. Produce a multi-trade section drawing at the clash location showing ALL services (structure, ductwork, pipework, cable trays, sprinklers) in their correct stack order with actual dimensions and clearances. Include: min 50mm below bottom-most service to ceiling void, 25mm between services, 50mm above services to structural soffit. The CSS reveals the true void depth required and enables a data-driven decision on whether to lower the ceiling or redesign the services.

**Checks:**

- [ ] All services included in the section, not just the clashing ones
- [ ] Minimum clearances shown at every service interface
- [ ] Controlling service identified (the one driving the minimum void depth required)
- [ ] CSS reviewed and signed off by all trade contractors before ceiling is installed

### 2. Lower ceiling to accommodate services stack

**Effort:** Medium. **VO risk:** Yes (formal variation / redesign likely).

Lower the ceiling to the minimum achievable headroom. Calculate: structural soffit – services stack depth – min 50mm – ceiling board + finish = finished ceiling level (FCL). Verify resulting headroom meets code minimums. Update the RCP: lighting layout, diffuser positions, and sprinkler pendant positions all change when the ceiling is lowered.

**Checks:**

- [ ] Headroom code compliance check at revised ceiling level
- [ ] RCP reissued with revised ceiling level and all ceiling-mounted elements updated
- [ ] Sprinkler deflector-to-ceiling clearance maintained at revised level (min 25mm, max 75mm typical)

### 3. Reduce service sizes or reroute dominant service out of zone

**Effort:** High.

Identify the service causing the maximum void depth requirement. Explore: HVAC duct resizing (higher velocity, smaller cross-section — check noise NR criteria); pipe rerouting to a less congested ceiling zone; cable tray splitting. Each must be validated by the responsible MEP engineer.

**Checks:**

- [ ] MEP engineer to validate each resizing option against system performance
- [ ] Duct velocity and NR rating confirmed after resizing
- [ ] All services confirmed within void with clearances after resizing

## Validate before acting

- [ ] Measure actual structural soffit level — slab tolerances can vary ±15mm from design
- [ ] Confirm FFL and FCL from architectural drawings for this space
- [ ] List all services in the zone with their actual dimensions (including insulation) and clearance requirements

## Escalate immediately if

- Multiple zones across the floor plate are failing void depth — this is a design coordination failure at project level that requires architect + MEP lead to resolve, not individual clash resolution
- Headroom is already at minimum code — ceiling cannot be lowered; MEP rerouting is mandatory and the MEP lead must redesign the zone

## Downstream risk if unresolved

Ceiling coordination issues discovered after finishes are installed require ceiling tiles to be removed, services modified, and tiles replaced. This is one of the most common defect callbacks in commercial fitout. Ensure the coordinated section is signed off by all trades before ceiling tile installation begins.
